# compile

Synthesize week's work into team communications.

## Seven-Step Process

### Step 1: Historical Context

Query for two most recent weekly learnings files to identify unresolved threads, prior commitments, established tone.

### Step 2: Daily Note Analysis

Read all daily notes from past 7 days, extracting:
- What actually happened (not planned)
- Conversations that shifted thinking
- Decisions
- Breakthroughs
- Surprises

### Step 3: Strategic Context

Load company, project, personal workflow context files to understand which events connect to larger strategic questions.

### Step 3.5: Idea Evolution Tracing

Use backlinks and search to find ideas appearing across multiple days, showing thinking evolution.

### Step 4-6: External Signals

Review:
- Social media posts (past 7 days)
- Calendar entries
- Recent messages

For decisions, engagement patterns, team conversations.

### Step 7: Terminal Output

Synthesize into structured format without creating files.

## Output Template

```
WEEKLY LEARNINGS PREP -- Week [X], [Year]

FROM LAST EMAIL (threads to continue):
- [Unresolved items and promises]

CANDIDATE TOPICS (5-8 ranked by depth):
[#]. [Topic] -- [Project]
- What happened: [Specific events]
- The insight: [Non-obvious value]
- Source: [Daily notes/meetings/posts]

CONNECTING THREAD:
[Overarching theme tying specifics together]

OPERATIONAL UPDATES:
[Decisions, changes, completions]

SUGGESTED STRUCTURE:
[3-4 email sections recommendation]
```

**Key Principle:** "Best weekly learnings come from noticing patterns writer was too close to see."

Use `obsidian` CLI for daily note searches.
